# Acceptance Criteria

High-level product acceptance, not per-feature details only.

## AC-001 - User Management
The application is accepted when:
- users can register
- users can log in
- invalid credentials are rejected

## AC-002 - Core Workflow
The application is accepted when:
- users can create a record
- users can edit it before submission
- submitted records are visible in history

## AC-003 - Administration
The application is accepted when:
- admin users can view all records
- admin users can update statuses
